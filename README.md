## Features

- Interactive Solar System with 6 training planets
- Research Frameworks (10 cards with copy-to-clipboard)
- Power Free Skills (8 skills with prerequisites)
- Command Center with activity log
- Dark minimal theme
- Mobile responsive

## Training Planets

| Planet | Focus | Exercises |
|--------|-------|-----------|
| Foundation | Basic patterns | 45 |
| Strength | Progressive development | 120 |
| Skills | Complex movements | 85 |
| Mobility | Joint health | 60 |
| Power | Explosive movements | 40 |
| Endurance | Cardiovascular | 35 |
